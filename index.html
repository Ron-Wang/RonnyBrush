输入格式示例

题目：下列哪个不是JavaScript的基本数据类型？
类型：single
选项：A. String
B. Boolean
C. Object
D. Number
答案：C
解析：JavaScript的基本数据类型有：String、Number、Boolean、Null、Undefined、Symbol(ES6)、BigInt(ES10)。Object是引用类型。

题目：以下哪些是CSS盒模型的组成部分？
类型：multiple
选项：A. 内容(content)
B. 内边距(padding)
C. 边框(border)
D. 外边距(margin)
答案：A,B,C,D
解析：CSS盒模型由内容(content)、内边距(padding)、边框(border)、外边距(margin)四部分组成。

使用说明

  • 每道题目必须包含"题目"、"类型"、"选项"、"答案"字段
  • "解析"字段可选，如不提供将显示"暂无解析"
  • 支持题型：单选题(single)、多选题(multiple)、判断题(judge)、不定项选择题(indefinite)
  • 选项请用字母(A、B、C...)开头，每行一个选项
  • 多选题和不定项选择题答案请用逗号分隔(如：A,B,C)
